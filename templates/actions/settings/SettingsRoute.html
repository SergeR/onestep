Статус плагина для поселения

Чтобы плагин работал для выбранного поселения необходимо задать "ВКЛ"

Режим
- корзина и оформление заказа

Плагин работает вместо стандартной корзины и всех шагов оформления заказа

- только оформление заказа

Плагин работает вместо оформления заказа, стандартная страница корзины плагином не изменяется

Определение города по IP-адресу

Плагин будет пытаться определить адрес покупателя по его IP-адресу.
Плагин может определить страну, регион и город.

Маска для телефона

Ввод телефонного номера по маске.

Маска телефонного номера

Для номеров России используется маска +7(###)###-##-##

Использовать проверку заполнения обязательных полей формы
Подтверждение заказа всегда на виду
Подсказки для полей DaData API

Использовать серсис DaData.ru для создания подсказак при заполнение полей адреса.
При регистрации необходимо подтвердить email иначе подсказки могут не работать

DaData API-ключ

API-ключ, который можно скопировать из личного кабинета сервиса DaData.ru

Подсказки для полей ФИО

Подсказки для заполнения имени, фамилии, отчества

Подсказки для поля E-mail
Подсказки для полей Адреса

Подсказки для заполнения улицы, города, региона, страны

Только для ПК

Плагин работает только для настольных компьютеров и отключается для мобильных устройств.
Для мобильных работает стандартная корзина.

Шаблон

Шаблон темы дизайна, который использует плагин для вывода информации

Адрес страницы

Адрес страницы, по которому будет работать плагин.
Плагин не работатет с системными адресами такими, как cart/ и checkout/
